A Gaming as a Service (GaaS)

 It is a project that offers a game (or a part of a game) to be rented or performed on a client machine with several forms of factors, e.g. a game variant can be customized based on client settings. Such a platform provides an opportunity not only to publish game content dynamically but also update game elements in real-time. GaaS is a novel way for the gaming market to operate games in server-side. It should need a specific hardware requirement (i.e. IaaS), a platform is essential to run the game (i.e. PaaS) and, ultimately SaaS is required to deliver the game itself to the customer.
