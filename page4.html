ARTICLE HEADER

A serious game and virtual reality based education environment for bomb disposal robot.

This project aims to train bomb disposal trainee (i.e a police officer) using a serious game approach in a virtual reality-based training environment by using a set of bomb disposal scenarios. By providing a safe training environment, the primarygoal is to mitigate the risks of damaging the robot during the training process. The robot and its remote controller are both modeled in a virtual simulation environment in which a trainee could practice using terrains and mazes generated by the application. The environment aims to help trainees to practice with a virtual robot to learn real robotâ€™s behaviors, and ultimately assist the trainee for improving skills in his hand-eye coordination. In addition, a virtual environment can provide flexibility in training by providing a set of virtual scenarios.
